# Kealee Construction Documentation

This folder contains documentation for various modules and features of the Kealee Construction website.

## Available Documentation

### E-Commerce Module

**[ecommerce-module.md](./ecommerce-module.md)**

Complete documentation for the store front functionality including:
- Architecture overview
- Feature descriptions
- Data structures
- Cart management
- Integration points for backend
- Future enhancements

## Purpose

This documentation serves to:
1. **Maintain Context:** Preserve knowledge about implementation decisions
2. **Onboard Developers:** Help new team members understand the codebase
3. **Plan Future Work:** Document integration points and enhancements
4. **Reference Guide:** Quick reference for module functionality

## Adding New Documentation

When adding a new module or feature:
1. Create a new markdown file in this directory
2. Follow the structure of existing documentation
3. Include:
   - Overview
   - Architecture/Structure
   - Features
   - Integration points
   - Future enhancements
4. Update this README to reference the new documentation

## Maintenance

Documentation should be updated when:
- New features are added
- Architecture changes
- Integration points are implemented
- Significant refactoring occurs
